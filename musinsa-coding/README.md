# Musinsa Coding

무신사 스타일 웹사이트 구현 프로젝트. Next.js 프론트엔드와 Express 백엔드로 구성되어 있으며, DB는 SQLite를 사용해 별도 서버 설치 없이 로컬에서 실행 가능하다.

## 실행 방법

### 사전 준비

- Node.js (LTS)
- pnpm

### Frontend

```bash
cd frontend
pnpm install
pnpm dev
```

`http://localhost:3000`

### Backend

```bash
cd backend
pnpm install
pnpm dev
```

`http://localhost:3001`
